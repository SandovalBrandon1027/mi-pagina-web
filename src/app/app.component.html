Mi Portafolio

Para finalizar mi carrera en Tecnología en Desarrollo de Software, realicé un trabajo de integración curricular que consistió en desarrollar un e-commerce para la venta de artesanías. El proyecto fue implementado utilizando JavaScript para la parte del Backend y React para el Frontend. Además, para la gestión de productos y el registro de usuarios se utilizó MongoDB Atlas.

Link de la aplicación web: https://artesanias-jimenez.vercel.app
